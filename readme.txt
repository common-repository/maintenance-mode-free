== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for "Maintenance Mode Free"
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go to Maintenance to configure the plugin

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select maintenance-mode-free.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
6. Go to Maintenance to configure the plugin

= Using FTP =

1. Download Maintenance Mode Free Plugin
2. Extract the maintenance-mode-free directory to your computer
3. Upload the maintenance-mode-free directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
5. Go to Maintenance to configure the plugin
